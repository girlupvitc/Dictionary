# dictionary

A single-page application for displaying definitions and pronunciations from a
Google Sheet.

### Contributing

1. Fork the repo, then clone your fork the repo
2. `cd dictionary`
3. Make changes
4. Build the bundle with `npm run build`
5. Test your changes and make a PR
